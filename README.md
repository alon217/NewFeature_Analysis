# Predictive Model Performance & Feature Impact Analysis

## Project Overview
This project evaluates the systemic impact of a major model update across a diverse customer base. [cite_start]The analysis quantifies performance improvements, identifies distribution patterns of the lift, and provides data-driven recommendations for model refinement.

## Analytical Methodology
The evaluation was conducted through a structured three-step process:
1. **Data Preparation**: Cleaning and structuring raw performance logs.
2. **Performance Comparison**: Calculating success rate metrics before and after the update.
3. **Statistical Analysis**: Utilizing Pareto distribution and correlation analysis to validate the results.

## Key Insights
* **System-Wide Success**: 96% of the customer base showed a measurable improvement in performance after the model update.
* **Portfolio-Wide Lift**: Pareto analysis revealed that 80% of the total improvement was distributed across 75% of the customers, indicating a systemic upgrade rather than a localized fix.
* **Impact on Low Performers**: Customers with the lowest baseline performance (20-30% accuracy) received the most significant boost, often gaining 40-60% in accuracy.
* **Anomalous Cases**: Identification of edge cases where limited baseline data created statistical noise, requiring further validation against broader historical samples.

## Technical Stack
* **Tools**: Python for data processing and statistical visualization (Pareto charts, Correlation plots).
* **Methods**: Statistical Hypothesis Testing, Data Aggregation, and Comparative Analysis.
